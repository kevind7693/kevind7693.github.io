Shabr Project

This project is about me trying to create a site trying to advertise a fiction hot air balloon company using bootstrap.

I created this project using a lot of tools offered by bootstrap. The 6 main tools I have used from bootstrap in this project is card, table, navbar, carousel, jumbotron, and lead. These components allowed me to create a modern looking wedsite thats completely responsive on any size devices. Even though bootstrap allowed my to create a modern looking wedsite with ease, there was still challenges that I faced throughout the creation of this project. One such challenge is that bootstrap sometimes break my css, so in order to slove this problem I relocated all the bootstrap code to be placed after the css code. I also started to write my css in a different file so the 2 doesn't interfere with on another.
